# Threat Feed Aggregator

Threat Feed Aggregator to platforma do agregacji, normalizacji, wyszukiwania i eksportu
Indicators of Compromise (IOC), zaprojektowana w podejściu database-first.

System integruje wiele źródeł TI, zapisuje dane w PostgreSQL i udostępnia je przez:
- REST API
- Web UI
- CLI
- eksporty w formatach systemów bezpieczeństwa
